Introduction to Data Vizualisation

R
Application Designing a visualization The use of visualization is pervasive in the scientific production and communication: explanatory diagrams, graphs describing statistics, new experimental data plotted against theoretical expectations, etc. In each case, the author of the visualization tries to lead readers to a specific point of view, by emphasizing some aspects of the data while toning down other aspects. Results may be informative, decisive or misleading. Can you apply the design principles we talked about in class?

Introduction to Reproducible Analysis

R
Reproducible Analysis Typical data analysis workflow looks like this: collect data put it in a file or spreadsheet or database run some analyses, written in various scripts, perhaps saving some intermediate results along the way or maybe always working on the raw data. create some plots or tables of relevant summaries of the data, and then go and write a report about the results in a text editor or word processor.
